# simple-core

极简单、轻量级开发框架。

##使用方式

添加仓库

	<repositories>
		<repository>
			<id>tianyl-repo</id>
			<name>tianyl-repo</name>
			<url>https://raw.githubusercontent.com/tianyl1984/maven-repo/master</url>
		</repository>
	</repositories>

添加依赖

	<dependency>
		<groupId>com.tianyl</groupId>
		<artifactId>simple-core</artifactId>
		<version>0.0.1</version>
	</dependency>
